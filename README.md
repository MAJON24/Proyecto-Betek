# ✈️ Volando con Datos  
## Rumbo a decisiones estratégicas  
### PROYECTO PARA LA AEROLÍNEA COMERCIAL AIRLSYM  

---

## 📖 Descripción

CRITERIO DATA es una consultora tecnológica enfocada en análisis de datos, ayudamos a las empresas a optimizar sus procesos.  

La aerolínea AIRLSYM requiere centralizar su información operativa y comercial en una sola plataforma que permita un análisis seguro y eficiente.  

Este proyecto integra modelado de datos, consultas SQL, análisis y visualización en Power BI para resolver una problemática real del negocio.

---

## 🎯 En busca del resultado

Diseñar e implementar un dashboard en Power BI que consolide la información de toda la aerolínea, con acceso segmentado por roles de negocio mediante seguridad dinámica desde la base de datos MySQL.

### 🔹 Objetivos específicos

- Soportar la base de datos suministrada por la aerolínea para el análisis  
- Definir las historias de usuario para responder preguntas de negocio  
- Implementar un sistema de seguridad por roles desde la base de datos  
- Estructurar un dashboard de 5 páginas con información segmentada  
- Establecer las bases para la creación de medidas DAX

## 🧾 Conclusiones

- Centralización de información: Único dashboard para toda la aerolínea  
- Gobernanza de datos: Cada rol accede exclusivamente a su información  
- Innovación: Manuales de usuario dentro del reporte
